PIVlab Wiki

Welcome to the PIVlab Wiki!

When running PIVlab, you have three choices. Typically, you decide for one of them:

  • You can run PIVlab as a separate App on your windows computer ("Stand-alone version")
  • If you have Matlab, you can run PIVlab as a Toolbox for your Matlab installation ("Toolbox version")
  • You can run PIVlab from your webbrowser with limited features ("Matlab online version")
The wiki has instructions for each of these choices.
